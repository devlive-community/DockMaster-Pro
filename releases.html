v1.3.0
April 16, 2026

🚀 New Features

  • Window preview panel on icon hover — live thumbnails of all open windows with click-to-focus, configurable padding, card spacing, and dismiss delay
  • Seven system detail plugins: CPU (per-core usage & hardware info), Memory (breakdown + swap), Disk (storage usage + read/write activity), GPU (per-GPU utilization & VRAM), Network (per-interface speed + built-in speed test), Thermal (SMC temperatures & fan speeds), and Battery Health
  • Process Monitor plugin — real-time process list with CPU and memory usage per process
  • Icon glow effect — breathing light glow for running and selected dock icons
  • Sparkle particle animation — particle burst effects for running and selected icons
  • Gradient theme support — Gradient Dusk and Gradient Ocean presets, plus new Neon Night and Frosted Glass built-in themes
  • Custom app icon management in the theme editor — replace per-app icons directly from settings
  • Drawer section drag-reorder — rearrange drawer sections by dragging within the dock
  • Replace / reset icon via context menu — right-click any dock icon to swap or restore its icon for the active theme
  • Menu bar mode redesign — per-item status rings with full plugin support in the menu bar
